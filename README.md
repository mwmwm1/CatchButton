# 버튼잡기게임(C# 코딩)

## 개요-C# 프로그래밍학습

C#을 활용하여 이벤트 기능을 사용해 마우스 커서를 피해 도망가는 버튼을 클릭하여 점수를 획득하는 게임입니다.

-핵심기능: mouseenter 이벤트로 버튼이 이동
          점수 시스템 구현
          클릭 성공시 버튼 크기 축소
          게임 오버시 재시작 기능 구현
          버튼과 상호작용시 효과음 재생
-화면구성: 메인 폼, 움직이는 버튼(RunningButton), 점수와 버튼 위치를 표시해주는 제목 바, 게임오버시 나오는 재시작 버튼
